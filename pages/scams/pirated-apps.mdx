# Free Paid Application

## How it works
1. Someone sends you a message offering you a paid application for free
2. You install the application
3. Your computer is now infected with malware

## I've already fallen for this scam
1. Uninstall the application
2. Run a virus scan on your computer
3. Change your passwords

## How do I avoid this scam?
- Don't install applications from untrusted sources
- If you're not sure if a source is trusted, do some research
- If it sounds too good to be true, it probably is
